## 🙌 기능소개
##### 사용한 공공데이터 api
- 한국관광공사_국문_관광정보_서비스
- 기상청_관광코스별_관광지_상세_날씨_조회_서비스
>1. 두개의 api를 활용하여 관광지, 숙박업소, 음식점 정보를 조회할 수 있음.
>2. 원하는 관광지를 자신만의 관광코스로 만들어서 게시판에 공유 할 수 있음.
>3. 찜하기, 방문완료 등을 체크하여 해당 관광지들의 정보를 마이페이지에서 조회 할 수 있음.
